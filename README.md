# scrapbox-textlint

## command

```shell
$ yarn run deploy -p <project-name> -f <import-file.json>
$ cat <import-file.json> | yarn run deploy -p <project-name>

yarn run --silent generate:code \
  --title foo --filename foo.js --file foo.js \
  --title bar --filename bar.js --file bar.js \
| yarn run deploy -p <project-name>
```
